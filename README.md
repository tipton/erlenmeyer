# erlenmeyer
neck - a collection of fundamental capabilities for Flask applications

***neck*** is a collection of exemplar materials that demonstrate various methods or
approaches for implementing fundamental capabilities of a Flask web application.

Fundamental Capabilities include:

1. Ability to log to multiple disk files (and stdout) in any module
2. Application landing page
3. User login page
4. Application "action" page
5. Logout function
6. Alignment of client and server cookie management
7. config.py setup
8. Password setting and re-setting
9. Non-session transactions from user
10. Server-sent events
